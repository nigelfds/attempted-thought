This is a sample post written in **Markdown**. Each file in `src/posts/`
becomes its own page at `/blog/<filename>/`, and the three most recent show up
on the home page automatically.

## Writing posts

Add a new `.md` file to `src/posts/` with this frontmatter at the top:

```
---
title: Your post title
date: 2026-06-15
summary: Shown in the blog list and on the home page.
---
```

Then write the body in Markdown — headings, lists, `inline code`, links, and
so on. Run `npm run build` and the new post appears.
